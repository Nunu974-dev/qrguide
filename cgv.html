Les présentes Conditions Générales de Vente (CGV) régissent la vente des services proposés par l'EURL Club Marketing (SIRET : 919 489 021 00015) via le site internet QRGUIDE.FR.

2. Description des services

QRGUIDE.FR propose les services suivants :

  • Pack Création : Création d'un guide numérique personnalisé accessible via QR code (paiement unique de 150€)
  • Abonnement Mensuel : Hébergement et maintenance du guide numérique (8€/mois)
  • Abonnement Annuel : Hébergement et maintenance du guide numérique (75€/an, soit une économie de 21€)
  • Plaques QR code : Plaques physiques en PVC premium avec QR code personnalisé (A4 : 45€, A5 : 35€)
  • Solution Hôtels : Solution personnalisée sur devis pour les établissements hôteliers
